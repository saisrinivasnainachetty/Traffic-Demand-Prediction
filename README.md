# 🚦 TrafficVision AI

An AI-powered Smart Traffic Demand Prediction System built using Machine Learning, Flask, LightGBM, and XGBoost.

---

## 📌 Overview

TrafficVision AI predicts road traffic demand using historical traffic data and machine learning models. It helps traffic authorities and smart city applications estimate congestion levels and improve traffic management.

---

- AI-based Traffic Prediction
- LightGBM & XGBoost Models
